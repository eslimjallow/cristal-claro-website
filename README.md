# Cristal Claro - Website

Website profesional para Cristal Claro, empresa de limpieza de cristales en Las Palmas de Gran Canaria.

## Estructura del Proyecto

- `index.html` - Página principal del sitio web
- `styles.css` - Estilos y diseño responsive
- `script.js` - Funcionalidades interactivas
- `README.md` - Este archivo

## Características

- ✅ Diseño moderno y profesional
- ✅ Totalmente responsive (móvil, tablet, desktop)
- ✅ Navegación suave entre secciones
- ✅ Animaciones al hacer scroll
- ✅ Optimizado para SEO
- ✅ Enlaces a WhatsApp integrados

## Personalización

### 1. Actualizar Número de WhatsApp

Busque y reemplace `34600000000` en `index.html` con su número real de WhatsApp (formato: código de país + número sin espacios ni símbolos).

Ejemplo: Si su número es +34 612 345 678, use `34612345678`

### 2. Agregar Imágenes

Reemplace los placeholders de imágenes:

1. **Imagen Hero** (Hero Section):
   - Busque `<div class="hero-image-placeholder">`
   - Reemplace con: `<img src="ruta/a/su/imagen.jpg" alt="Cristal Claro">`

2. **Imagen Why Choose Us**:
   - Busque el `<div class="image-placeholder">` en la sección "Por Qué Elegirnos"
   - Reemplace con: `<img src="ruta/a/su/imagen.jpg" alt="Servicio profesional">`

3. **Imagen Contacto**:
   - Busque el `<div class="image-placeholder">` en la sección "Contacto"
   - Reemplace con: `<img src="ruta/a/su/imagen.jpg" alt="Contacto">`

### 3. Colores Personalizados

Puede cambiar los colores en `styles.css` modificando las variables CSS en `:root`:

```css
--primary-color: #0066cc;      /* Color principal */
--primary-dark: #0052a3;       /* Color principal oscuro */
--secondary-color: #00a8e8;    /* Color secundario */
```

## Cómo Usar

1. Abra `index.html` en su navegador web
2. O despliegue los archivos en su servidor web

## Notas

- Asegúrese de tener imágenes optimizadas para web (formato JPG o PNG, tamaño razonable)
- Pruebe el sitio en diferentes dispositivos para verificar la responsividad
- Actualice el número de WhatsApp antes de publicar el sitio
